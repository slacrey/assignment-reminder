# 催孩子写作业

本地单机 Web MVP。家长可以添加孩子并绑定 QQ 号，创建作业并设置提醒时间。到点后，系统会自动生成一条模拟 QQ 提醒日志。

## 功能

- 添加孩子：姓名、QQ 号。
- 创建作业：孩子、标题、说明、提醒时间。
- 自动提醒：后台每 30 秒扫描到点作业。
- 模拟发送：不接真实 QQ，只写提醒日志。
- 管理页面：查看孩子、作业状态和提醒日志。

## 暂不支持

- 真实 QQ 登录或真实 QQ 消息发送。
- 家长账号、登录鉴权、多用户隔离。
- 公网部署、多设备同步。
- 重复提醒、完成确认、催促升级。
- 跨时区处理；MVP 使用本机本地时间设置提醒。

## 安装

```bash
uv sync
```

## 运行测试

```bash
uv run pytest
```

## 启动应用

```bash
uv run uvicorn app.main:app --reload
```

打开：

```text
http://127.0.0.1:8000
```

## 数据位置

默认 SQLite 数据库：

```text
data/assignment_reminder.sqlite3
```

`data/` 是本地运行数据目录，已加入 `.gitignore`。

## 验收流程

1. 打开本地管理页面。
2. 添加一个孩子和 QQ 号。
3. 创建一条一两分钟后提醒的作业。
4. 等待后台扫描。
5. 确认作业状态变为“已提醒”。
6. 确认提醒日志出现模拟 QQ 消息。
